# codereader

コードの簡単な動きを確認するサービスの企画用リポジトリです。

## 現在の状態

**未実装です。**

現時点ではREADME以外のソースコード、画面、テスト、ビルド設定は存在しません。そのため、現在のリポジトリだけではアプリの起動・動作確認・コードレビューはできません。

## 実装前に確定が必要な項目

- 対象とする言語（HTML/CSS/JavaScript、Pythonなど）
- コードを実際に実行するのか、処理の流れだけを可視化するのか
- ブラウザ内だけで完結させるのか、サーバーを使用するのか
- ユーザーが入力したコードをどのように隔離・制限するのか
- `preview`リポジトリのWebミニ実行機と統合するか、別サービスにするか

これらを確定してから、MVPの仕様書と安全な実行方式を決めて実装を開始します。
